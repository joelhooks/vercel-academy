### Troubleshooting Guide

Avoid common challenges:

<Callout type="warning" title="Debugging Cheatsheet">
	* **Rate Limits:** Check [OpenAI limits](https://platform.openai.com/docs/guides/rate-limits).
	Implement retries. * **Context Windows:** Long chats = token overflow. Consider truncating
	history. * **Tool Errors:** Add `console.log` in `execute()`. Verify parameters on the server. *
	**Schema Errors:** For improved outputs, refine `.describe()` or use more capable models.
</Callout>

	title="Exploring Different Model Providers"
	description="The AI SDK makes it easy to switch between different LLM providers."
	prompt="Try replacing OpenAI with another provider in one of the lessons. What changes are needed? How do the outputs differ?"
	steps={[
		"Install additional provider packages with `pnpm add @ai-sdk/anthropic` or `pnpm add @ai-sdk/google`",
		"Update imports: `import { anthropic } from '@ai-sdk/anthropic'` or `import { google } from '@ai-sdk/google'`",
		"Replace model specification: `model: anthropic('claude-3-opus-20240229')` or `model: google('gemini-1.5-pro')`",
		"Adjust any provider-specific parameters in your prompts",
		"Compare performance, output quality, and token usage across providers"
	]}
	code={`
// OpenAI
import { openai } from '@ai-sdk/openai';
const { text } = await generateText({
	model: openai('gpt-4o'),
	prompt: "Explain quantum computing in simple terms"
});

// Anthropic
import { anthropic } from '@ai-sdk/anthropic';
const { text } = await generateText({
model: anthropic('claude-3-opus-20240229'),
prompt: "Explain quantum computing in simple terms"
});

// Google
import { google } from '@ai-sdk/google';
const { text } = await generateText({
model: google('gemini-1.5-pro'),
prompt: "Explain quantum computing in simple terms"
});
`}
